# TRADING-STRATEGY-ANALYTICS-DASHBOARD
Power BI dashboard analyzing 5000+ trades across NIFTY50 stocks using a custom trading dataset extracted with Python.

Instead of using a public dataset, I extracted and processed my own trading data using Python and analyzed 5000+ trades across NIFTY50 stocks over a 3-year period.
